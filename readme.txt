Please use chrome to view this game.

Player needs to reach in the exit within 100s

Score is updated at the end of each level. Add L * 100 points for passing level L.
And also add X point for each second of remaining time, where you choose an appropriate value of X. where X = 1
Score is updated when a monster is shot – add Y points when this happens, you choose an appropriate value of Y where Y = 10
Score is updated when a good thing is touched – add Z points when this happens, you choose the value of Z where Z = 5
